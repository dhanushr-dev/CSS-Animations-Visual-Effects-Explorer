# ✨ CSS Animations & Visual Effects Explorer

A creative and interactive **CSS Animations & Visual Effects Explorer** built using **HTML5 and CSS3**. This project demonstrates modern CSS animations, transitions, transforms, and visual effects through practical examples, helping beginners learn how to create engaging and dynamic web interfaces.

## 🚀 Live Demo

🔗 https://css-animation-explorer.netlify.app/

## 📂 GitHub Repository

🔗 https://github.com/dhanushr-dev/CSS-Animations-Visual-Effects-Explorer

- 🎬 CSS Animations
- 🔄 CSS Transitions
- 🎯 CSS Transformations
- ✨ Hover Effects

- 🌟 Box Shadow & Text Shadow
